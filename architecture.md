# Video Transcoding Architecture

```
[ 👤 User ] ──(1. Upload "sample.mp4")──> [ 🌐 FastAPI ]
                                               │
                           ├── (2. 원본 저장) ├── (3. 작업 지시)
                           ▼                   ▼
     [ 🪣 MinIO ('videos' 버킷) ]        [ 🧠 Redis (Queue) ]
           └── sample.mp4                      │
                                               ├──> 📬 [ audio_queue ] ──┐
                                               └──> 📬 [ video_queue ] ──┼─┐
                                                                         │ │
                    ┌────────────────────────────────────────────────────┘ │
                    │                                                      │
                    ▼                                                      ▼
    [ 🎧 audio_worker (컨테이너 1) ]                      [ 🎬 video_worker (컨테이너 2) ]
      1. 원본 다운로드                                       1. 원본 다운로드
      2. 오디오만 추출 (m4a)                                 2. 비디오만 변환 (소리 없음)
      3. 버킷에 오디오 업로드 ───┐                           3. 오디오 업로드될 때까지 대기 ⏳
                                 │                           4. 버킷에서 오디오 다운로드
                                 ▼                           5. ⚡️ 비디오 + 오디오 병합 (Copy)
               [ 🪣 MinIO ('transcoded' 버킷) ]               6. 버킷에 최종본 업로드 ─┐
                     └── 📂 sample/                                                   │
                           ├── 🎧 audio.m4a <─(비디오 워커가 가져다 씀)               │
                           │                                                          │
                           ├── 🎬 1080p.mp4 (소리+영상 완벽) <────────────────────────┤
                           ├── 🎬 720p.mp4  (소리+영상 완벽) <────────────────────────┤
                           └── 🎬 360p.mp4  (소리+영상 완벽) <────────────────────────┘
```
